About me ( Web Portfolio นี้ผมใช้ AI เเละ ความสามารถของกระผมทำขึ้นมาครับ )

สวัสดีครับ ผมชื่อ 'เด็กชายวสกร ภูมิประโคน' ผมเป็นนักเรียนโรงเรียนอนุบาลกัลยาประสิทธิ์ ที่มีความสนใจในการเข้าเรียนที่ โรงเรียนเตรียมอุดมศึกษาน้อมเกล้านครราชสีมา ห้องโครงการห้องเรียนเทคโนโลยีดิจิทัล (IDGT) ผมเริ่มเขียน code ตั้งเเต่ชั้นประถมศึกษาปีที่ 6 จนถึงปัจจุบัน ผมเริ่มศึกษาการเขียนเกมใน Platform Roblox Studio โดยใช้ภาษา Lua เเละเริ่มขยับมาศึกษาเกี่ยวกับทำบอท Discord โดยใช้ภาษา Python เเละ JavaScript ณ ปัจจุบันนี้ผมก็เริ่มศึกษาภาษาเขียนโค้ดอื่นๆมากขึ้น เพื่อที่จะพัฒนาตนเองยิ่งขึ้น


นอกเหนือจากการศึกษาการเขียนโปรเเกรม เเล้วกระผมยังสนใจในเรื่องของดนตรี เเละกีฬา
